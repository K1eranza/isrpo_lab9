# Лабораторная работа №9: Адаптивная верстка с медиа-запросами

## Цель работы
Изучить принципы адаптивной верстки и медиа-запросов для создания страниц, которые корректно отображаются на всех устройствах.

## Выполненные задания
1. Создание базового адаптивного макета с медиа-запросами
2. Тестирование адаптивности на разных устройствах
3. Создание адаптивной навигации (вертикальное/горизонтальное меню)
4. Разработка Flexbox-макета с тремя колонками

## Теория
**Медиа-запросы (Media Queries)** — техника CSS, которая позволяет применять разные стили в зависимости от характеристик устройства (ширина экрана, ориентация и т.д.).

### Синтаксис
```css
@media (условие) {
    /* стили для этого условия */
}
